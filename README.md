# 서버 프로그램 구현

본 저장소는 **서버프로그램 구현 과목**의 강의안과 실습 코드를 포함합니다.

Spring Boot 기반의 메인 서버와  
Node.js 기반의 배치/워커 서버를 구성하여,  
**서버프로그램의 기본 구조와 역할 분리**를 실습을 통해 학습합니다.

---

## 사용 기술

- WSL2 + Ubuntu 24.04
- Nginx, Spring Boot, Node.js
- MySQL 8.x
- Git

---

## 학습 내용 요약

- Controller / Service / Repository 구조 이해
- SQL 직접 작성 기반 데이터베이스 연동
- 서버 로직에서의 트랜잭션 개념
- 공통 응답 및 예외 처리 구조
- 메인 서버와 배치/워커 프로세스의 역할 분리
