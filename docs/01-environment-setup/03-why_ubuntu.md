# 리눅스 배포판 소개 ( Ubuntu )
> 리눅스 배포판은 공통된 리눅스 커널에 패키지·도구·정책을 묶어 배포한 운영체제이다.

## 1. 리눅스 배포판이란 무엇인가

앞 장에서 서버 프로그램은 리눅스 환경에서 운영된다는 것을 배웠다.   
하지만 리눅스는 하나의 OS 이름이 아니라, **여러 배포판(distribution)** 으로 나뉘어 있다.

즉, 리눅스는 다음과 같은 구조를 가진다.

* 공통된 리눅스 커널
* 배포판마다 다른 패키지 구성과 정책

그래서 서버를 만들 때는
"리눅스를 쓴다"가 아니라
**"어떤 리눅스 배포판을 쓸 것인가"** 를 선택해야 한다.

---

## 2. 서버에서 배포판 선택이 중요한 이유

리눅스 배포판마다 다음 요소가 다르다.

* 기본 패키지 구성
* 패키지 관리자
* 업데이트 정책
* 보안 패치 주기
* 문서와 자료의 양

서버 환경에서는 **"가장 안정적이고 자료가 많은 배포판"** 이 중요하다.

---

## 3. 대표적인 리눅스 서버 배포판

서버 환경에서 자주 언급되는 배포판은 다음과 같다.

* Ubuntu Server
* Debian
* CentOS
* Amazon Linux

---

## 4. 왜 이 강의에서 Ubuntu를 사용하는가

이 강의의 목적은
리눅스 자체를 깊게 배우는 것이 아니다.

목표는 다음과 같다.

* 서버 프로그램 구조 이해
* 실제 서버와 유사한 환경 경험
* 실무에서 쓰는 흐름 익히기

이 목적에 가장 잘 맞는 선택이
**Ubuntu**다.

---

## 정리

* 리눅스에는 여러 배포판이 있다
* Ubuntu는 서버·교육 환경 모두에 적합하다
* 이 과정에서는 Ubuntu 24.04 LTS를 사용한다
