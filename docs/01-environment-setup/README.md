# 개발환경 구축

이 파트에서는
서버 프로그램 구현을 위한 **개발 환경을 처음부터 끝까지 준비**한다.

목표는
"툴을 설치했다"가 아니라,
**왜 이 환경이 필요한지 이해하고 스스로 다시 구성할 수 있는 상태**가 되는 것이다.

---

## 이 파트에서 다루는 것

* 서버 프로그램이 무엇인지에 대한 전체 그림
* Linux / Ubuntu를 사용하는 이유
* WSL2 기반 Ubuntu 서버 환경 구성
* 서버용 기본 패키지 설치
* Linux 서버 기초 감각 익히기

---

## 문서 구성 (폴더 내부 번호 기준)

1. 서버프로그램 구현 과정 소개 (`01-intro.md`)

3. Ubuntu를 사용하는 이유 (`03-why_ubuntu.md`)
4. WSL2 + Ubuntu 24.04 환경 구성 (`04-wsl2_ubuntu_setup.md`)

6. 필수 서버 패키지 설치 (`06-basic_packages.md`)
7. 리눅스 서버 기초 맛보기 (`07-linux_server_basics.md`)

---

## 이 파트의 학습 결과

이 파트를 마치면 학생은 다음을 할 수 있어야 한다.

* Windows에서 Ubuntu 서버 환경을 직접 구성할 수 있다
* Linux 서버에 접속하고 기본 명령을 사용할 수 있다
* 서버 프로그램을 실행하기 위한 준비 상태를 스스로 점검할 수 있다
