# 강의 안내

이 디렉터리는 **서버프로그램 구현 과목의 강의 문서**를 모아둔 공간이다.

각 문서는 개별적인 설명서가 아니라,
**서버프로그램의 구조와 흐름을 단계적으로 이해하기 위한 학습 자료**로 구성되어 있다.

---

## 학습 목표

이 문서들을 통해 다음을 이해하는 것을 목표로 한다.

* 서버프로그램이 무엇을 책임지는지 설명할 수 있다
* 하나의 요청이 서버 내부에서 어떻게 처리되는지 이해한다
* 역할 분리(Controller / Service / Repository)의 이유를 설명할 수 있다
* 오래 걸리는 작업을 분리해야 하는 이유를 이해한다
* 메인 서버와 배치/워커 처리 흐름의 차이를 구분할 수 있다

---

## 학습 안내

강의는 상위 개념부터 구조와 흐름 순서로 진행된다.

문서를 읽을 때는
구현 방법보다 **왜 이런 구조가 필요한지**에 집중하는 것을 권장한다.
